# Shopping System (Java + JDBC + MySQL)

A simple user-friendly shopping system built using Java, JDBC, and MySQL.

## Features
- Secure Login
- View Products
- Add to Cart
- Place Order
- MySQL Database Storage

## Technologies Used
- Java
- JDBC
- MySQL
- HTML/CSS

## How to Run
1. Import project into IDE or just use command line.
2. Run SQL script from sql/shoppingdb.sql in MySQL.
3. Update DB username/password in DBConnection.java.
4. Compile and run ShoppingApp.java.
